# DefaultPack
Default pack for TomShelbyBot

## Features
- Poll reactions
- Mute
- Clap (gag) mute
- Commands to have fun with

## Commands
```
Итак.. вот список того, что я умею.. (32): 

/all
- Упомянуть всех, кого бот знает.

/beat
- Изрезать...

/bomb
- Заспамить чатик упоминаниями юзера

/check (алиасы: /lookup)
- Основная информация из сведений семьи Шелби.

/coin (алиасы: /throw, /flip)
- Подбросить монетку

/f
- F in chat

/gag (алиасы: /кляп, /clapmute, /muteclap)
- Вставить кляп. Именно кляп! Особый вид мута

/go
- Отправит опрос со слов 'Погнали ...'.

/goose
- Закинуть гуся.

/help@night_tom_bot (алиасы: /start@night_tom_bot)
- Помощь по коммандам бота

/id (алиасы: /chatid)
- Узнать id чата

/info
- Основаня информация о боте

/mute
- Под угрозой расстрела сообещния запретить человеку писать.

/pcheck
- Проверить наказания

/random (алиасы: /number)
- Получить рандомное число (диапазон тоже можно указать).

/say
- Сказать что-то от имени бота

/stickermode (алиасы: /sticks, /togglest)
- Запрещать извергам слать стикеры.

/summon
- Отправить призыв прописать комманду

/toxic
- Поставить алерт на токсика

/unmute (алиасы: /pardon)
- Размутить. Выдать.. Право.. Голоса

/unsummon
- Сбросить призыв прописать комманду

/wall
- Построить непробиваемую стенку.
```
